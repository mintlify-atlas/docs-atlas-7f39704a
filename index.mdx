# Siigo Corprecam Scraper

Automate your Siigo accounting workflows with intelligent browser automation. This tool streamlines purchase data entry by scraping external APIs and automatically filling Siigo document support forms using Playwright.

## Key Features

<CardGroup cols={2}>
  <Card title="Automated Data Entry" icon="robot" color="#1e40af">
    Automatically creates Siigo document support entries from purchase data
  </Card>
  <Card title="Multi-Company Support" icon="building" color="#60a5fa">
    Handles both Corprecam and Reciclemos companies with separate workflows
  </Card>
  <Card title="Resilient Automation" icon="rotate" color="#1e40af">
    Built-in retry logic ensures reliable execution even with network issues
  </Card>
  <Card title="Ngrok Integration" icon="tunnel" color="#60a5fa">
    Expose local server via ngrok for seamless webhook integration
  </Card>
</CardGroup>
